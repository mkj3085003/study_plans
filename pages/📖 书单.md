- # 书单
  - 说明
    - 每本书用一个块记录
    - 常用属性：type:: book, status::, progress::, author::, started::, finished::

- ## 书单列表
  - 书籍示例
    - [[示例书：刻意练习]]
      - type:: book
      - status:: to_read
      - progress:: 0%
      - author:: 安德斯·艾利克森

- ## 待读
  - {{query (and (property type book) (property status to_read))}}

- ## 在读
  - {{query (and (property type book) (property status reading))}}

- ## 已读
  - {{query (and (property type book) (property status done))}}
